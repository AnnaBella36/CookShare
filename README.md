# CookShare

CookShare is an iOS application that helps users discover, view, and save recipes.  
It was developed as a learning project to demonstrate modern iOS development practices with a clean architecture and reactive data flow.

---

## Main Features
- Recipe search and filtering  
- Add recipes to favorites  
- Offline data storage using Core Data  
- User authentication and auto-login  
- Asynchronous image and data loading  
- Minimal and responsive SwiftUI interface  

---

## Tech Stack
Swift 5, SwiftUI, Combine, MVVM, Core Data, URLSession (async/await), XCTest
